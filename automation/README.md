# Automation Framework Overview

This folder demonstrates a sample automation framework for a Drupal-based application.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)

## Structure
- features/ → Gherkin test scenarios
- step-definitions/ → Step implementations
- runner/ → Test execution setup

## Coverage
- Content workflow validation
- Role-based access control
- Anonymous user restrictions

## Purpose
This automation setup demonstrates how Drupal workflows and permissions can be validated using structured BDD automation.
